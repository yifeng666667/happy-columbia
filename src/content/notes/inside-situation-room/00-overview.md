## 这门课在学什么

2011年5月1日，一张照片震惊了全世界：美国总统奥巴马、副总统拜登、国务卿希拉里，以及一屋子神情凝重的国家安全团队，挤在白宫一个不大的会议室里，盯着屏幕实时观看猎杀本·拉登的行动直播。没有人说话，空气凝固。这个房间，就是 **情境室（Situation Room）**。

这门课研究的，正是这种时刻：当一个国家最重要的决策，必须在极短时间、极高不确定性、后果不可逆的条件下做出——**谁在那个房间里？他们怎么想？他们怎么决策？他们为什么有时候会犯下改变历史走向的错误？**

这不只是政治学，更是心理学、社会学、历史学的交叉地带。它用真实案例告诉你：世界格局的走向，不是由某种神秘的"历史规律"决定的，而是由一群有偏见、有情绪、受国内政治和官僚体系制约的真实人类，在特定的房间里，根据不完整的信息做出的决定。

## 为什么要学

如果你想读懂今天的新闻——乌克兰战争、中美博弈、中东乱局、贸易战——你需要这门课提供的**思维工具**：不是记住哪年发生了什么，而是能看穿"为什么这个领导人会做出这个决定，背后的逻辑链是什么"。

更重要的是：**这门课让你对权力保持清醒**。你会发现，改变世界历史走向的决策，往往不是深思熟虑的结果，而是群体压力、认知偏见、国内政治和信息失真共同作用的产物。理解这一点，是任何有志于政策、外交、金融或商业领域的人必须建立的世界观。

## 学完之后能做什么

- 看到一则外交新闻，能自动问出：**这个决策背后谁在受益？国内政治如何约束了领导人的选项？信号传递是否被对方正确理解了？**
- 理解"贸易战不只是经济问题"——它是国内政治、选举逻辑、地缘博弈的综合产物
- 在分析投资或政策时，能加入"**领导人的心理偏见与决策失误**"这个维度，而不只是看经济数据

## 三副眼镜：特朗普 / 奥巴马 / 拜登

这门课的最大特色，是用**三种不同的权力哲学**来解读同一个世界。把它们理解成三副眼镜——戴上哪一副，看到的"问题"和"解决方案"截然不同。

| 维度 | 🔴 特朗普 | 🔵 奥巴马 | ⚪ 拜登 |
|---|---|---|---|
| **世界观** | 零和博弈：别人赢就是我输 | 多边主义：共同规则让所有人受益 | 民主 vs 威权：这是我们时代的根本对立 |
| **决策风格** | 直觉驱动，反建制，交易思维 | 审慎分析，听顾问，追求共识 | 经验优先，制度忠诚，节奏偏慢 |
| **权力来源** | 个人魅力 + 不可预测性 | 叙事能力 + 多边协调 | 盟友网络 + 机构信誉 |
| **信誉策略** | 不可预测即威慑（"疯子溢价"） | 讲原则但执行力不足（叙利亚红线） | 可预测但被对手认为软弱 |
| **对华政策** | 最大压力 + 关税武器 | 接触 + 竞争并存（"战略再平衡"） | 联盟框架内竞争（"derisking"） |
| **外交遗产** | 退群（TPP/WHO/伊朗协议）+ 亚伯拉罕协议 | 伊朗核协议 + 亚太再平衡 + 杀死拉登 | 支持乌克兰 + 修复盟友关系 |
| **最大争议** | 损害民主制度与盟友信任 | 叙利亚红线食言 + 无人机滥用 | 阿富汗撤军混乱 + 对华政策延续性 |

> **如何用这张表**：每读一个session，都可以回来问：这件事发生在三位总统身上，各自会怎么处理？这种比较会让你对权力本质的理解深入十倍。

## 课程主线

![白宫战情室](https://images.unsplash.com/photo-1501466044931-62695aada8e9?auto=format&fit=crop&w=900&q=80)
*▲ 华盛顿特区——全球最重要的决策中心*

这门课按照**"人 → 思 → 谈 → 选 → 控"**五幕结构展开：

**第一幕：人**（Sessions 1-2）——密室里坐的是什么样的人？他们的身份、背景、性别、阶层，如何在做出决策之前就预先决定了"可见的选项"？

**第二幕：思**（Sessions 3-4）——这些人怎么思考？危机状态下大脑会发生什么？情报信息是如何经过层层失真，以一种扭曲的面貌出现在领导人面前的？

**第三幕：谈**（Sessions 5-6）——当选项确定后，如何与对手沟通？外交是什么，信号是什么，威胁是什么？为什么有些威胁有效，有些只是空话？

**第四幕：选**（Sessions 7-8）——最终，是干涉还是不干涉？是妥协还是施压？国内政治如何在最后时刻绑架了本来"合理"的外交决策？

**第五幕：控**（Sessions 9-10）——谁在控制信息？社交媒体如何改变了外交的底层逻辑？而贸易战，是工具还是武器？

## Session 速览

|---|---|---|
| 01 | 领导者真的重要吗？ | 是人改变了历史，还是历史选择了人？ |
| 02 | 谁在密室里？ | 房间里的人决定了看得见的选项 |
| 03 | 危机中的大脑 | 为什么聪明人在压力下做蠢事？ |
| 04 | 情报：失真的信息 | 领导人相信的"现实"和真实现实有多大差距？ |
| 05 | 外交的艺术 | 秘密谈判 vs 公开表态，哪种外交有效？ |
| 06 | 红线与空话 | 威胁说出口，没有执行，代价是什么？ |
| 07 | 干涉的两难 | 出手还是不出手，都是错的？ |
| 08 | 国内政治的枷锁 | 总统真的在为国家利益决策吗？ |
| 09 | 信息战与民主侵蚀 | 威权主义者如何在信息战中占据上风？ |
| 10 | 贸易战：2018 vs 2025 | 同一剧本，为何第二幕代价更大？ |
