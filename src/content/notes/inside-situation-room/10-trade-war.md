## 这节课在回答什么

两场贸易战，同一个发起人，相隔七年——它们表面上都是"对华加关税"，但背后的逻辑、规模、战略目标，以及中美双方的博弈方式，已经截然不同。

**2018年**：特朗普的第一轮贸易战，是一场用关税换谈判筹码的商人式博弈，最终以"第一阶段协议"收场，双方都能对国内交代。

**2025年**：特朗普回来了，但世界已经变了——这一轮是一场关于**科技主导权**和**全球供应链重构**的系统性对抗，没有明显的退路。

要理解这两场战争，必须先理解**美中经济关系四十年的来龙去脉**——因为任何贸易摩擦，都不是凭空产生的。

## 一、背景：美中四十年的经济缠绕

### 1.1 "超级全球化"如何让两国绑在一起

1978年，邓小平推动改革开放。1992年，中国明确建立社会主义市场经济体制。2001年，**中国加入WTO**——这是改变全球经济格局最重要的单一事件之一。

入世之后，美中经济进入了一个被经济学家称为**"超级全球化（Hyperglobalization）"**的时代：

- 美国跨国公司把制造业搬到中国，利用廉价劳动力降低成本
- 中国工厂生产的商品，通过沃尔玛等零售渠道，压低了美国通货膨胀
- 中国积累了大量美元外汇储备，购买美国国债，压低了美国长期利率

这个系统，**对美国消费者有利**（便宜的商品），**对美国跨国公司有利**（高利润），**对中国制造业有利**（高就业、高增长）——

但**对美国工人不利**：大量制造业岗位消失，铁锈带（Rust Belt）的工业城市空洞化。

### 1.2 "中国冲击"的学术证据

2016年，经济学家大卫·奥托尔（David Autor）和同事发表了一篇极具影响力的论文，量化了**"中国冲击（China Shock）"**：

- 1999-2011年，美中贸易导致美国制造业损失了大约**200-240万个工作岗位**
- 影响最集中在特定的工业社区（家具、纺织、电子）
- 这些社区的工人，**没有**像经济学教科书预测的那样"转型到其他行业"——他们大多长期失业

**政治影响**：失业的工业工人，集中分布在宾夕法尼亚、密歇根、威斯康星、俄亥俄——恰好是2016年和2024年的关键摇摆州。

特朗普的贸易战，从政治经济学的角度说，是在回应一个**真实存在的经济伤害**——尽管关税是否是最好的解决方式，存在巨大争议。

### 1.3 从"接触"到"竞争"：战略叙事的根本转变

奥巴马时代，美国对华主流战略叙事是：**接触政策（Engagement Policy）**——通过经济融合，中国会逐渐走向市场化、民主化，变成"负责任的利益相关者"。

这个叙事在2015-2017年开始系统性崩溃：
- 中国没有走向政治自由化，习近平反而在集权
- 中国的产业政策（"中国制造2025"）被美国视为国家补贴对美国科技产业的直接威胁
- 南海军事化，"一带一路"被视为地缘扩张

2017年，特朗普政府的《国家安全战略》正式把中国定义为**战略竞争者（Strategic Competitor）**——这标志着40年接触政策的终结，也是两场贸易战的真正起点。

## 二、第一场贸易战（2018-2020）：商人的逻辑

### 2.1 导火索：301调查

2018年3月，美国贸易代表办公室（USTR）发布了一份**301调查报告**，指控中国：
- 强迫外资企业技术转让
- 系统性窃取美国知识产权
- 国家补贴扭曲市场竞争
- 歧视性地限制美国企业进入中国市场

报告估计，这些行为每年造成美国损失**2250亿至6000亿美元**。

### 2.2 关税升级的完整时间线

| 日期 | 事件 |
|---|---|
| 2018年3月 | 美对进口钢铁（25%）铝（10%）加征全球关税，中国是目标之一 |
| 2018年7月6日 | 美对340亿美元中国商品加征25%关税（第一批），中方立即对等报复 |
| 2018年8月23日 | 美对160亿美元中国商品加征25%关税（第二批），中方对等 |
| 2018年9月 | 美对2000亿美元中国商品加征10%关税（第三批），中方报复550亿美元 |
| 2018年12月 | G20布宜诺斯艾利斯，习近平与特朗普会面，达成"90天休战" |
| 2019年5月 | 谈判破裂，美将第三批关税从10%升至25% |
| 2019年8月 | 美宣布对3000亿美元中国商品加征关税，随后暂缓部分 |
| 2020年1月15日 | **"第一阶段协议"签署** |

### 2.3 第一阶段协议：谁赢了？

协议内容：
- 中国承诺两年内**额外**购买2000亿美元美国商品（比2017年基准多）
- 中国承诺加强知识产权保护
- 美国同意不对已有关税追加新关税，部分关税从25%降至7.5%

**实际执行**：中国只完成了承诺购买量的约57%——主要原因是2020年新冠疫情打乱了全球贸易。

**谁"赢"了？**
- 从特朗普的角度：他对国内展示了"强硬对华"，拿到了中国对美国农产品的大规模采购承诺（对他的农业州选民有直接价值），同时通过最终签协议展示了"我能成交"
- 从中国的角度：结构性问题（知识产权、产业补贴、市场准入）基本没有实质改变，付出的代价是采购承诺（最终未完全履行）
- 从历史的角度：这场贸易战**没有解决任何根本性问题**，反而把中美双方都推向了更深的战略互疑

### 2.4 2018贸易战的核心特征

1. **以关税为筹码，以谈判为目的**：特朗普从一开始就想要"成交"，不是想永久脱钩
2. **规模有限**：即使在最高峰，涉及商品覆盖约3600亿美元，美国对华商品平均关税从3.1%升至约20%
3. **美国盟友没有跟上**：欧盟、日本、韩国基本上没有参与对华关税协调
4. **供应链基本未变**：虽然有企业开始把部分产能迁出中国，但主体供应链格局变化有限
5. **"Decoupling"（脱钩）是话语，不是现实**

## 三、第二场贸易战（2025年至今）：系统性重构

### 3.1 回来之前，世界已经变了

特朗普2025年重返白宫时，继承的是拜登四年对华政策留下的遗产：

拜登并没有取消特朗普的关税（这一点让很多人意外）——实际上，他在关键领域**加码了**：

- **2022年10月**：《芯片与科学法案》（CHIPS Act）签署，向美国本土半导体制造投入527亿美元补贴，同时颁布对华**出口管制**——限制向中国出口先进芯片（≤14nm）和芯片制造设备
- **2022年8月**：《通货膨胀削减法案》（IRA），4000亿美元绿色能源补贴，核心条件是在北美生产——隐性地把中国排除在补贴链之外
- **2023年**：美国联合荷兰（ASML）、日本（Tokyo Electron）共同管制对华高端光刻机出口——这是多边技术封锁的开始

**关键理解**：拜登的对华政策，叫做**"小院高墙（Small Yard, High Fences）"**——不是全面脱钩，而是在**关键科技领域**（半导体、AI、量子计算）建立高壁垒，其他领域维持正常贸易。

特朗普2025年面对的，已经是一个中美之间战略竞争框架已经成形的世界。

### 3.2 2025年4月"解放日"：史无前例的关税冲击

2025年4月2日，特朗普宣布对**几乎所有国家**征收"对等关税（Reciprocal Tariffs）"——

但对中国，是不同级别的对待：

| 时间 | 关税动作 |
|---|---|
| 2025年2月 | 对中国商品先加征10%（理由：芬太尼问题），随后再加10%，累计+20% |
| 2025年4月2日 | 宣布对中国商品再加征34%"对等关税" |
| 2025年4月4日 | 中国宣布对美国商品加征34%反制关税 |
| 2025年4月7日 | 特朗普威胁：如果中国不撤销报复，将再加征50% |
| 2025年4月9日 | 对**其他国家**90天暂停，对**中国**额外加征至145% |
| 2025年4月 | 中国宣布对美国商品关税升至**125%** |

到2025年4月中旬，**美国对华商品平均关税已超过145%**（叠加之前各轮）。

这是什么概念？平均145%的关税，在实践上**几乎等于贸易禁运**——大多数普通商品在这个关税水平下，没有进口的经济价值。

### 3.3 中国的反制：不只是关税

2025年，中国的反制策略，比2018年复杂得多：

**关税对等**：125%对等关税（中国对美出口品的经济依赖比2018年低，承受力更强）

**稀土与关键矿物出口管制**：
- 2025年4月，中国宣布限制**稀土元素**出口许可——这触及美国航空、国防、电动车、半导体制造的核心产业链
- 中国控制全球约60%的稀土开采和约85%的稀土加工——这是不对称的战略筹码

**实体清单**：将美国企业列入出口管制实体清单，限制向其出口关键原材料

**非正式抵制**：中国航空公司暂停部分波音飞机接收订单，影响估值数十亿美元的合同

**算法与宣传**：中国媒体（包括海外社交账号）大规模输出"美国挑起贸易战损害全球经济"的叙事

### 3.4 对全球供应链的影响

2018年的贸易战，是一个警告——全球供应链开始"China+1"战略（在越南、墨西哥、印度增设备份产能）。

2025年的贸易战，是一次**强制性重组**：

**"Friendshoring（友岸化）"加速**：苹果加快把iPhone组装迁往印度；越南、泰国、墨西哥成为制造业迁入地

**但"去中国化"的极限**：
- 中国制造业占全球制造业产出的约28%，没有任何单一国家可以在近期替代
- "组装在越南，零件来自中国"——很多"中国+1"实际上只是在中国和最终目的地之间加了一个"转口"
- 145%关税下，即使如此，部分商品的竞争力仍然存在（中国的完整产业链、工人技能、基础设施）

## 四、2018 vs 2025：系统性对比

![美中博弈](https://images.unsplash.com/photo-1526778548025-fa2f459cd5ce?auto=format&fit=crop&w=900&q=80)
*▲ 两场贸易战，同一个发起人，但世界已经截然不同*

| 维度 | 2018年贸易战 | 2025年贸易战 |
|---|---|---|
| **根本性质** | 用关税换谈判筹码 | 系统性技术与贸易脱钩 |
| **关税规模** | 峰值约3600亿美元商品，平均关税~20% | 几乎全部商品，关税145%+ |
| **中国依赖度** | 中国对美出口占GDP约3% | 中国对美出口依赖有所下降（多元化），但科技领域更深度捆绑 |
| **盟友协调** | 基本单边行动 | 对盟友也同时加关税（"对等关税"），然后90天暂停，呈现出"孤立中国"的潜在框架 |
| **科技维度** | 几乎没有 | 核心战场——芯片、AI、量子、稀土 |
| **中国反制工具** | 主要是对等关税 | 关税+稀土管制+实体清单+非正式抵制+叙事战 |
| **退出路径** | 清晰（"成交"） | 模糊——没有明确的"第一阶段协议2.0"框架 |
| **对全球经济影响** | 局部可控（供应链开始转移） | 系统性冲击（通胀预期重燃、全球衰退风险上升） |
| **美国国内政治背景** | 2018年中期选举压力 | 2024年已赢得选举，2025年执行"结构性变革"议程 |

## 五、三副眼镜：解读贸易战

### 🔴 特朗普：这是美国赢回失去的世纪

特朗普的贸易战哲学，有其内在一致性，但常被误解：

**他不是自由贸易的信徒**——他从1980年代起就认为美国被"不公平贸易"伤害了（他1987年就在《纽约时报》刊登广告批评美国贸易政策）。

他的框架：
- **关税是收入**：美国消费者为进口品付关税，这是对美国国库的贡献，而不是"损失"（这个观点经济学家基本不认同，因为关税成本最终由进口商和消费者承担）
- **不平衡就是被剥削**：中国对美贸易顺差=中国在"赢"，必须纠正
- **谈判中的压力**：关税是迫使对方坐到谈判桌的工具，不是目的本身

**批评**：这个框架忽略了比较优势、全球价值链的复杂性，以及关税最终会压低美国消费者的实际购买力。

### 🔵 奥巴马：多边机制，不是单边关税

奥巴马的对华贸易策略，是通过**TPP（跨太平洋伙伴关系协定）**来制衡中国——建立一个把中国排除在外的贸易规则体系，用规则来围堵，而不是用关税来对抗。

TPP被特朗普在2017年1月上台后第一周就签字退出。

奥巴马的判断：单边关税会损害美国盟友关系，无法持续，而多边规则体系才是长期可持续的竞争工具。

**批评**：TPP需要国会批准，在美国国内的政治阻力极大（进步派反对，因为担心损害工人权益；特朗普的民粹主义利用了同样的民怨）。

### ⚪ 拜登："小院高墙"vs 特朗普的"大院高墙"

拜登选择的方式，被概括为**"Invest, Align, Compete"**——投资（CHIPS法案、IRA）、联盟（与欧盟、日韩协调）、竞争（关键领域出口管制）。

他明确不走全面关税路线——认为这只会推高美国通货膨胀，损害美国消费者，而不能实质性改变中国的产业政策。

2025年，当特朗普把关税推到145%，批评者指出了一个讽刺：
- 2025年4月，美股大幅下跌，通胀预期上升，市场对衰退的担忧急剧升温
- 这些都是拜登政府试图避免的经济代价

**但支持者反驳**：有时候短期的经济痛苦，是换取长期结构性变革的必要代价。

## 六、投资含义：从贸易战读懂宏观

贸易战不只是政治新闻，它直接影响市场和投资判断：

**供应链重构的赢家**：东南亚制造业（越南、泰国、马来西亚）、墨西哥、印度——大量制造业在"友岸化"过程中迁入

**半导体产业的重组**：美国政府大力补贴的台积电（亚利桑那）、英特尔、三星在美工厂——这是一个需要十年尺度来看的长期投资故事

**通货膨胀预期**：关税是供给侧冲击，它推高进口商品价格。145%的对华关税，影响约5000亿美元的美国进口——其通胀影响可能相当于联储额外加息1个百分点

**中国资产的再定价**：持续的战略不确定性，意味着外资在中国资产上的**地缘政治风险溢价**不会消失——不是"中国不好"，而是不可预测性需要定价

**美元地位的长期问题**：如果美国用美元体系作为武器（关税、制裁），持有美元储备的国家会开始寻找替代——这是一个缓慢但深刻的结构性压力

## 七、为什么重要：这门课的终点，也是理解世界的起点

这节课，是整个"权力密室"课程的收尾——也是最接近当下的一节。

把这门课的所有框架汇聚到贸易战上：

- **领导者重要吗（Session 01）**：是的——两场贸易战，都是特朗普主导的，与他个人的商人哲学高度一致。但结构性矛盾（中美科技竞争、制造业空心化）是独立于领导者的
- **密室里的人（Session 02）**：贸易政策是由谁主导的？2025年，纳瓦罗（Peter Navarro，强硬鹰派）、贝森特（Scott Bessent，财政部长，相对温和）、莱特希泽（Robert Lighthizer，贸易谈判代表）之间的张力，直接影响政策的激进程度
- **红线与信誉（Session 06）**：美国在90天暂停后是否会真正执行？还是最终成为另一次"划而不守"？中国的稀土出口管制是否只是筹码？——这些问题的答案，将决定下一步的博弈方向
- **双层博弈（Session 08）**：特朗普在国内的政治压力（通胀上升、市场下跌、中期选举）将是约束他加码的最重要变量——最终，国内政治往往比国际博弈更能决定出口

贸易战不会结束——它会演变。理解它的结构，是读懂未来十年全球经济和地缘政治格局的基础。
