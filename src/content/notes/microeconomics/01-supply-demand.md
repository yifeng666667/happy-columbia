## 本文概述

这篇笔记讲的是整个微观经济学的地基：**供给（Supply）与需求（Demand）**。几乎所有你在新闻里看到的价格现象——北京学区房价格、2024 年国际油价、禽流感之后鸡蛋变贵——背后都可以用这一套框架解释。更重要的是，这个框架还能告诉你：当政府试图"管住"价格时，会发生什么。

![供需曲线基本图示](https://upload.wikimedia.org/wikipedia/commons/thumb/7/7a/Supply-and-demand.svg/400px-Supply-and-demand.svg.png)

*供需曲线交叉决定均衡价格与均衡数量——这是整个微观经济学的起点。*

## 一、需求：为什么价格越高买的越少

### 1.1 需求曲线的直觉

**需求（Demand）** 描述的是：在其他条件不变的情况下，消费者愿意并且有能力在不同价格水平下购买某商品的数量。

最核心的规律叫 **需求定律（Law of Demand）**：价格越高，需求量越低；价格越低，需求量越高。这是一条向右下方倾斜的曲线。

为什么？两个直觉：
- **替代效应**：某样东西变贵了，你会去买替代品（猪肉贵了，多买点鸡肉）
- **收入效应**：价格上涨等于你的实际购买力下降，自然少买

### 1.2 需求量的变动 vs 需求的变动

这是很多初学者混淆的地方，必须分清楚：

- **需求量的变动（Change in Quantity Demanded）**：价格变了，导致你在同一条需求曲线上移动。比如油价从 7 元/升涨到 9 元/升，你开车少了。
- **需求的变动（Shift in Demand）**：需求曲线整体移动。价格没变，但其他因素变了。

哪些因素会让需求曲线整体右移（需求增加）？

| 因素 | 例子 |
|---|---|
| 收入增加 | 经济好，人们愿意多买车 |
| 偏好改变 | 健身热潮让运动鞋需求上升 |
| 相关商品价格变化 | 猪肉暴涨，替代品牛肉需求上升 |
| 预期价格上涨 | 传说房价要涨，现在买更多 |
| 人口增加 | 城镇化推动房屋需求上升 |

### 1.3 真实案例：一线城市房价

2020-2021 年北京、上海、深圳核心地段的房价持续上涨，可以用需求侧的多重因素叠加来解释：
- 疫情期间低利率，购房融资成本下降
- "双减"政策出台前，优质学区房的稀缺溢价被进一步强化
- 人口持续净流入，长期需求支撑

这三个因素同时作用，把需求曲线往右推，价格随之上涨——这不是投机炒作，是供需模型的基本预测。

## 二、供给：为什么价格越高生产越多

### 2.1 供给曲线的直觉

**供给（Supply）** 描述的是：在不同价格水平下，生产者愿意并且能够提供的商品数量。

**供给定律（Law of Supply）**：价格越高，供给量越多。这是一条向右上方倾斜的曲线。逻辑很简单——价格高意味着更高的利润，更多企业有动机进入，原有企业也愿意扩大产量。

### 2.2 哪些因素会让供给曲线移动

| 因素 | 方向 | 例子 |
|---|---|---|
| 生产成本上升（原材料、人工） | 左移（供给减少） | 铁矿石涨价，钢铁供给减少 |
| 技术进步 | 右移（供给增加） | 光伏组件技术进步，成本大降，供给量大幅上升 |
| 相关商品价格变化 | 视情况 | 玉米价格上涨，农民减少大豆种植面积 |
| 政策/监管 | 视情况 | 环保限产，高污染行业供给缩减 |
| 预期价格下跌 | 右移（现在多卖） | 生产者预期明年产能过剩，今年加速出货 |

### 2.3 真实案例：2024 年油价波动

2024 年全球油价（WTI 原油）从年初约 75 美元/桶，中间多次波动。影响供给侧的关键力量：
- **OPEC+ 减产协议**：沙特主导的产量管控，把供给曲线往左推，支撑价格
- **美国页岩油产量创纪录**：技术进步使美国成为最大产油国，供给持续扩张，压制价格上涨空间
- **地缘政治风险溢价**：中东局势紧张时，市场预期供给中断，价格短期上冲

这就是两股力量博弈——OPEC+ 压缩供给、美国页岩油扩大供给——导致油价在某个区间反复震荡的微观逻辑。

## 三、均衡：供需如何决定价格

### 3.1 市场均衡

**均衡（Equilibrium）** 是供给量等于需求量的那个点，对应的价格叫**均衡价格（Equilibrium Price）**，对应的数量叫**均衡数量（Equilibrium Quantity）**。

直觉上理解：
- 如果市场价格高于均衡价格 → 供过于求（Surplus）→ 生产者降价促销 → 价格下降
- 如果市场价格低于均衡价格 → 供不应求（Shortage）→ 消费者竞价 → 价格上升
- 价格自动向均衡点收敛

```
价格偏高 → 过剩 → 价格下降 → 趋向均衡
价格偏低 → 短缺 → 价格上升 → 趋向均衡
```

这个"价格自动调节"的机制，就是亚当·斯密所说的"**看不见的手（Invisible Hand）**"。

### 3.2 均衡的变化：供需曲线移动后会怎样

| 变化 | 均衡价格 | 均衡数量 |
|---|---|---|
| 需求增加（曲线右移） | 上涨 | 增加 |
| 需求减少（曲线左移） | 下跌 | 减少 |
| 供给增加（曲线右移） | 下跌 | 增加 |
| 供给减少（曲线左移） | 上涨 | 减少 |

### 3.3 真实案例：禽流感与鸡蛋价格

2023-2024 年多国爆发禽流感，大批蛋鸡被扑杀。供给曲线急剧左移，在需求相对稳定的情况下，鸡蛋价格上涨。美国部分地区鸡蛋零售价一度涨至每打 5-6 美元，较正常水平翻了几倍。

这是一个教科书式的"供给冲击（Supply Shock）"案例：**供给短期内无法快速恢复**（蛋鸡从雏鸡养到产蛋需要数月），因此价格上涨持续了相当长一段时间。

## 四、价格管制：好心可能办坏事

### 4.1 价格上限（Price Ceiling）

**价格上限（Price Ceiling）** 是政府规定的最高价格，通常低于均衡价格，目的是让商品更"买得起"。

但结果往往适得其反：
- 价格被压低 → 需求量增加，供给量减少 → **供不应求（Shortage）**
- 生产者减少供给，消费者排队抢购
- 黑市（Black Market）出现，实际成交价格可能高于管制前

**典型案例**：租金管制（Rent Control）。纽约、旧金山长期存在的租金管制，表面上帮助了现有租客，但抑制了房东改善和出租房产的动力，造成租赁供给长期萎缩，反而让没有租到管制房的人面临更高的市场租金。中国很多城市在 2021-2022 年试行的房租指导价，也面临类似的矛盾。

### 4.2 价格下限（Price Floor）

**价格下限（Price Floor）** 是政府规定的最低价格，通常高于均衡价格，目的是保护生产者或劳动者的收入。

结果同样有副作用：
- 价格被抬高 → 需求量减少，供给量增加 → **供过于求（Surplus）**
- 商品卖不掉，产生积压

**典型案例**：最低工资（Minimum Wage）。最低工资是劳动力的价格下限。如果最低工资高于均衡工资，企业会减少雇人（需求量减少），同时愿意工作的人变多（供给量增加），结果是失业。当然，现实中最低工资对就业的影响比理论预测的复杂，但这个基本逻辑成立。

另一个案例：欧盟对农产品的价格补贴，长期导致农产品过剩，产生了著名的"黄油山"和"葡萄酒湖"——大量农产品无法在市场上销售，政府被迫出资收购。

## 五、为什么重要

供需模型看起来简单，但它提供了一套**反直觉的思维工具**。很多政策的出发点是好的，但效果却南辕北辙——限价导致短缺、收税导致两败俱伤、补贴导致过剩——这些结果用供需框架都能预测到。

后面几篇笔记都建立在这个基础上：Session 2 会讲弹性（价格变动有多大影响），Session 3 会讲消费者怎么做决策（需求背后的微观机制），Session 4 会讲企业的供给决策，Session 5-6 会分析不同市场结构和市场失灵。供需模型是整个系列的起点。

## 六、2026 年视角：供需框架下的中国经济

2025-2026 年的中国经济，几乎每条主要新闻都能用供需框架拆解：

**需求侧**：居民消费信心仍在恢复期，房地产财富缩水压制了消费意愿（需求曲线左移）。政府试图通过消费券、以旧换新补贴等方式直接推动需求右移，但效果受限于"资产负债表衰退"心理。

**供给侧**：中国制造业产能在光伏、电动车、锂电池领域严重过剩，供给曲线大幅右移导致价格持续下降。这对消费者是好事，但对企业利润率和行业健康度构成挑战——典型的"丰收悖论"。

**价格管制的现实版**：2025 年多地出台的二手房"指导价"、新能源车补贴退坡后的价格竞争，都可以用本文的价格上限/下限框架来分析其副作用。
