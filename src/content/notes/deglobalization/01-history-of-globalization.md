## 本文概述

今天我们觉得"全球化"是理所当然的——超市里有新西兰的奇异果，手机里有台湾的芯片，衣柜里有孟加拉的T恤。但全球贸易的扩张并不是一条直线，它经历了两个世纪的起伏——自由贸易的黄金年代、两次世界大战的毁灭性中断、大萧条的保护主义灾难、以及战后在美国主导下的缓慢重建。理解这段历史，是理解今天逆全球化浪潮的起点。

![19世纪的全球贸易网络——帆船时代的港口](https://upload.wikimedia.org/wikipedia/commons/f/f4/Clipper_Ship_Southern_Cross_Leaving_Boston_Harbor_1851.jpeg)

*1851年的波士顿港：19世纪是第一次全球化的黄金时代，廉价的海运、金本位和自由贸易思想共同推动了前所未有的跨国贸易扩张。*

## 一、19世纪：第一次全球化

### 1.1 自由贸易思想的诞生

18世纪末到19世纪初，经济学作为一门学科正在成形，围绕"贸易是否有益"的辩论异常激烈。

亚当·斯密（1776年《国富论》）和大卫·李嘉图（1817年比较优势理论）提出了经济学史上最持久的论点之一：

> **即使一个国家在所有商品的生产上都比另一个国家效率低，两国之间的自由贸易仍然能让双方都受益。**

这就是**比较优势（Comparative Advantage）**——不是"谁更擅长做什么"，而是"谁放弃得更少"。英国擅长纺织，葡萄牙擅长酿酒，即使葡萄牙的纺织工人也比英国更熟练，两国专注于各自的比较优势再交换，总产出仍然更高。

### 1.2 谷物法的废除（1846年）

理论是一回事，政治是另一回事。19世纪上半叶英国最激烈的贸易辩论围绕**《谷物法》（Corn Laws）**展开——这是一系列保护英国地主阶级利益的农产品进口关税。

| 支持谷物法（保护派） | 反对谷物法（自由贸易派） |
|---|---|
| 英国贵族和大地主 | 新兴工业资本家和城市工人阶级 |
| 论点：保护国内农业和粮食安全 | 论点：高粮价推高工资成本，损害工业竞争力 |
| 代表：托利党保守派 | 代表：Richard Cobden、Anti-Corn Law League |

**1846年，谷物法被废除**——这是自由贸易思想在英国取得的标志性胜利，也开启了近70年的自由贸易扩张时代。

### 1.3 金本位下的贸易繁荣

1870年代至1914年是**古典金本位（Classical Gold Standard）**时代，也是第一次全球化的高峰：

**金本位的运作逻辑：**
- 各国货币与黄金挂钩，汇率事实上固定
- 相当于"通胀目标为零"——金本位下物价长期稳定（如果黄金供给跟不上经济增长，甚至会出现通缩）
- 固定汇率消除了汇率风险，极大降低了国际贸易成本

**推动贸易繁荣的四大因素：**

1. **运输成本骤降**：蒸汽船和铁路的普及大幅压低了跨国运输成本
2. **通讯技术革命**：跨大西洋海底电缆（1866年）让商业信息可以即时传递
3. **金本位的汇率稳定**：降低了跨国交易的不确定性
4. **英国的自由贸易政策示范效应**：作为当时的头号经济大国，英国单方面推行自由贸易

到1913年，全球贸易占GDP的比重达到了此后近80年都未能再次企及的水平。

### 1.4 第一次全球化的赢家与输家

这段历史已经揭示了一个延续至今的核心张力：

> **贸易的整体收益是真实的，但分配效应同样是真实的——总有人因贸易而受损。**

19世纪的**赫克歇尔-俄林理论（Heckscher-Ohlin）**的核心预测：

- 一个国家会出口密集使用其丰富要素的商品
- 贸易会提高丰富要素的价格，压低稀缺要素的价格
- 在19世纪：新大陆（美国、澳大利亚）土地丰富 → 出口农产品，地价上升；欧洲劳动力丰富 → 出口制成品
- **但欧洲地主利益受损**（廉价进口粮食压低了地价）——这就是为什么他们拼命游说恢复关税

## 二、两次世界大战与贸易的崩溃

### 2.1 一战：全球化的第一次死亡

1914年爆发的第一次世界大战不仅是军事灾难，更是全球贸易体系的毁灭性打击：

- 交战国之间的贸易完全中断
- 航运被征用于军事运输
- 各国实施严格的资本管制和贸易管制
- 金本位被暂停（各国需要印钱为战争融资）

**战后的问题更加棘手：**
- 巨额战争债务压在战胜国和战败国身上
- 德国的天文数字赔款（凯恩斯在《和约的经济后果》中尖锐批评这一安排）
- 各国急于恢复金本位——但这是一个致命的错误

### 2.2 "回到原价"的灾难

战争期间各国大量印钱，物价普遍上涨了50%甚至更多。但战后各国政治家执迷于回到战前的金本位平价——这意味着需要把物价压回战前水平。

**这在经济上意味着什么？**

物价下降50%不是自然发生的——它需要一场严重的经济衰退。紧缩的货币政策 → 通缩 → 实际债务负担加重 → 企业破产 → 大规模失业。

英国1925年在丘吉尔（时任财政大臣）主导下恢复金本位，经济学家凯恩斯发表了著名的小册子《丘吉尔先生的经济后果》，警告这将造成灾难——但被忽视了。结果正如凯恩斯预言：英国经济陷入长期低迷，1926年爆发了全国性大罢工。

### 2.3 大萧条与保护主义的恶性循环

1929年华尔街崩盘引发了大萧条，全球贸易体系在随后几年内几乎彻底崩溃。

**斯穆特-霍利关税法案（Smoot-Hawley Tariff Act, 1930）** 是这段历史中最臭名昭著的政策：

- 美国将约2万种进口商品的关税提高到历史最高水平
- 超过1000名经济学家联名反对——这可能是经济学界最大规模的联名抗议
- 其他国家立刻报复：英国1932年推出"帝国特惠"（只对英联邦国家零关税），法国、德国等纷纷竖起贸易壁垒

| 指标 | 1929年 | 1932-33年 |
|---|---|---|
| 全球贸易总额 | 基准 | 下降约65% |
| 美国失业率 | ~3% | ~25% |
| 物价水平 | 基准 | 下降约25%（严重通缩） |

经济研究估计，**贸易政策本身可能导致了全球贸易崩溃的一半以上**——不仅仅是经济衰退的被动结果，保护主义主动加速了崩溃。

> **大萧条的真正教训不是"贸易太多了"——而是宏观政策（过紧的货币政策 + 金本位的束缚）出了根本性的错误，保护主义只是让一切变得更糟。**

### 2.4 从贸易崩溃到世界大战

贸易崩溃、大规模失业、社会动荡——这些经济灾难直接催生了1930年代的政治极端主义。德国的纳粹党、意大利的法西斯在这片经济废墟上崛起。

> 某种意义上，斯穆特-霍利关税法案与二战之间有一条清晰的因果链：保护主义 → 贸易崩溃 → 经济萧条加剧 → 政治极端主义 → 战争。

这个教训深刻地影响了战后国际秩序的设计者们。

## 三、战后重建：从布雷顿森林到WTO

### 3.1 美国主导的战后秩序

二战结束后，以美国为首的战胜国以惊人的远见设计了一整套国际经济治理框架——这也许是历史上最具善意的霸权行为：

| 机构 | 成立时间 | 核心功能 |
|---|---|---|
| **IMF（国际货币基金组织）** | 1944年（布雷顿森林会议） | 维护汇率稳定，为国际收支困难的国家提供贷款 |
| **世界银行** | 1944年 | 为发展中国家提供发展融资 |
| **GATT（关贸总协定）** | 1947年 | 通过多轮谈判逐步降低关税，建立贸易规则 |

最初的设想是建立一个完整的**国际贸易组织（ITO）**——但美国参议院拒绝批准（美国的孤立主义传统在此再次显现）。GATT 作为一个更有限的替代方案应运而生。

### 3.2 GATT 的核心原则

GATT 建立了战后贸易体系的基本规则：

1. **最惠国待遇（MFN）**：你不能对不同贸易伙伴差别对待——给一个国家的关税优惠必须给所有成员（例外：自由贸易区和关税同盟）
2. **互惠（Reciprocity）**：关税减让必须是双向的——你降我也降
3. **透明性**：贸易政策必须公开、可预测
4. **争端解决机制**：有规则就需要执行——GATT（后来的WTO）提供了国际贸易争端的仲裁平台

### 3.3 缓慢但稳定的贸易恢复（1950s-1980s）

战后贸易恢复远比人们想象的缓慢：

- 全球贸易直到1970年代才真正恢复到1913年的相对水平
- GATT 经历了八轮关税谈判（从1947年的日内瓦回合到1994年的乌拉圭回合）
- 每一轮都扩大了参与国数量、降低了平均关税

**1950-1980年代的贸易增长有一个重要特点**：它主要发生在发达国家之间——美国、欧洲、日本相互之间的贸易。这些国家的要素禀赋（资本、劳动力技能水平）相对接近，贸易的分配效应（谁赢谁输）相对温和。

> **这一点至关重要**——当1990年代贸易开始涉及要素禀赋截然不同的国家（发达国家 vs 中国/印度的低技能廉价劳动力），分配效应骤然放大，政治反弹随之而来。这是下一节的核心内容。

### 3.4 GATT 到 WTO 的升级（1995年）

1995年，GATT 正式升级为**世界贸易组织（WTO）**——一个更正式、更有执行力的国际机构。WTO 不仅覆盖货物贸易，还扩展到服务贸易、知识产权等领域。

WTO 最重要的时刻之一：**2001年中国正式加入 WTO**——这为随后的"超级全球化"铺平了道路，也埋下了我们今天面对的许多问题的种子。

## 四、核心教训

| 教训 | 历史依据 |
|---|---|
| **贸易扩张不是自然规律，而是政策选择的结果** | 1846年废除谷物法、战后 GATT 体系——都需要政治意愿 |
| **保护主义在经济困难时期最具吸引力，但效果最差** | 大萧条期间的关税战让所有人更穷 |
| **金本位 ≈ 通胀目标为零——在通缩环境下是灾难** | 1920年代英国、大萧条的全球通缩 |
| **贸易崩溃可以催生政治极端主义** | 1930年代的法西斯崛起 |
| **战后秩序的设计者们从两次大战中学到了教训** | IMF、世界银行、GATT 的建立 |
| **发达国家之间的贸易政治阻力较小** | 1950-1980年代的温和贸易增长 |
