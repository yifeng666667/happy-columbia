## 一、宏观 vs 微观：两种观察经济的视角

### 1.1 微观经济学在看什么

**微观经济学（Microeconomics）**的主角是个体：一个消费者怎么做选择，一家企业如何定价，一个市场里供需怎么出清。它的核心问题是"**某一件具体的事为什么发生**"——比如苹果手机为什么卖那么贵，最低工资上调之后餐厅会不会裁员，垄断企业的利润从哪里来。

### 1.2 宏观经济学在看什么

**宏观经济学（Macroeconomics）**把镜头拉远，看整个经济体：一个国家的总产出怎么计量，整体价格水平为什么会涨，就业率为什么会波动，政府和央行能做什么、不能做什么。它的核心问题是"**整个系统表现如何，以及谁在驱动它**"。

两者的区别，用一个比喻来说：微观是研究一棵树如何生长，宏观是研究整片森林的气候变化。

### 1.3 为什么不能只看其中一个

宏观和微观并不是割裂的。2022 年美联储加息（宏观政策），直接推高了个人房贷利率（微观影响），让数百万家庭的购房计划被迫搁置。中国房地产行业的债务危机（微观层面若干开发商的破产），最终演变成拖累整体经济增速的宏观问题。学宏观，必须有微观的底，因为宏观现象最终都是无数个体决策的加总。

## 二、宏观经济学的三个核心问题

所有宏观经济学的研究，最终都在围绕三个基本问题打转：

```
问题一：经济增长（Growth）
  → 一个国家的财富总量是如何增加的？
  → 为什么有的国家富裕，有的国家贫困？
  → 短期波动和长期趋势背后的驱动力是什么？

问题二：通货膨胀（Inflation）
  → 为什么物价会上涨？上涨多少算"健康"？
  → 恶性通胀和通缩，哪个更可怕？
  → 央行能控制通胀吗？代价是什么？

问题三：失业（Unemployment）
  → 为什么总有人找不到工作？
  → 哪些失业是"正常"的，哪些是需要干预的？
  → 就业和通胀之间有没有必然的权衡？
```

这三个问题不是孤立的，它们深度交织。经济增速过快，通胀往往随之而来；为了压通胀而加息，又会让企业缩减招聘，失业率上升。理解这三者的互动，就是理解宏观经济分析的核心。

## 三、为什么这对投资者和普通人都重要

### 3.1 对投资者

不理解宏观，就很难看懂市场为什么动。

- **2022 年全球股债双杀**：美联储把利率从 0.25% 一路加到 5.5%，十年期美国国债收益率从 1.5% 飙升到 5%。这不只是"债券跌了"，而是整个资产定价体系被重置——所有未来现金流都用更高的折现率来估值。如果不懂货币政策，这场最大规模的资产价格重调就只是看不懂的"暴跌"。
- **中国房地产的宏观逻辑**：房地产贡献了中国 GDP 的近 20%（如果算上上下游产业链），当这个行业出问题，财政收入减少、消费信心下滑、银行坏账上升，多米诺骨牌是有既定顺序的。看懂这个顺序，才能判断政策会在哪个节点出手。

### 3.2 对普通人

宏观政策就是你的生活环境。

- **通货膨胀就是隐性税**：钱放在账户里不动，如果通胀是 5%，你一年就损失了 5% 的实际购买力。
- **利率决定你的房贷成本**：央行每一次加息或降息，都直接影响着几亿个家庭的月供金额。
- **就业政策的背后是宏观判断**：政府决定要不要刺激经济、扩大财政支出，背后的逻辑都是宏观分析——尽管执行效果未必和预期一致。

## 四、这个系列的结构与主线

本系列共 8 篇内容笔记，加上这篇总览，共 9 篇。前 6 章是教科书框架，后 2 章是把框架落到当下两个最重要的经济体——中国与美国——的实战拆解。

```
先定义"怎么衡量经济体"
  → Session 1：GDP（测量经济体量）

然后看"经济运行中会出现哪些问题"
  → Session 2：通货膨胀（价格失控）
  → Session 3：失业（劳动力市场失衡）

再看"谁来调节这些问题、怎么调节"
  → Session 4：货币政策（央行的工具箱）
  → Session 5：财政政策（政府的收支）

接着看"一国经济如何嵌入全球系统"
  → Session 6：国际贸易与汇率

最后把框架落到当下两个最关键的经济体
  → Session 7：中国宏观 2026（百年变局下的拆解）
  → Session 8：美国宏观 2026（软着陆、再加杠杆、AI 叙事）
```

每一个 session 都是独立可读的，但它们共享一套底层逻辑：**经济系统不是自我平衡的，它会出问题，政策是人为干预系统的工具，而干预本身也会带来副作用。**
