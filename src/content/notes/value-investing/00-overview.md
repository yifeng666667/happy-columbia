## 这门课在学什么

价值投资（Value Investing）是一套完整的投资哲学与方法论——它不是简单的"买便宜股票"，而是一种理解商业、理解市场、理解人性的思维方式。这门课从价值投资的起源讲起，经过估值方法的系统训练，到投资大师的实践智慧，最终上升到文明演进与财富本质的宏观视角。

课程的知识框架可以这样理解：

```
Ben Graham 奠基（三大基石）
  → Greenwald 估值体系（四层估值法）
  → Buffett/Munger 的质量革命（能力圈 + 优质公司）
  → 行为金融与投资纪律（为什么知易行难）
  → 李录的文明视角（财富的本质 + 全球价值投资）
  → 价值投资在中国（现代化、资本市场与机遇）
```

每一讲都在回答同一个根本问题的不同面向：**如何在一个充满噪音的市场里，找到真正有价值的东西，买入并持有？**

## 为什么要学

你可能听过巴菲特、芒格、李录这些名字，也大概知道"长期持有""安全边际"这些词。但如果你只是碎片化地了解这些概念，遇到真实的市场波动时，你大概率拿不住。

价值投资不是一套公式，而是一种**认知体系**。它帮你回答几个关键问题：
- 一家公司到底值多少钱？（估值框架）
- 这个价格是贵了还是便宜了？（安全边际）
- 我真的理解这家公司吗？（能力圈）
- 市场暴跌 50% 的时候我能不能不动？（投资纪律）
- 什么才是真正的财富？（财富观）

对想进入投资行业、或者想管理好自己资产的人来说，这些问题的答案决定了你未来几十年的财富轨迹。

## 学完之后能做什么

- **独立分析一家公司的价值**：用 Greenwald 四层估值法（资产价值 → 盈利能力价值 → 特许经营价值 → 增长价值），给出有逻辑的估值判断
- **区分"便宜"和"价值陷阱"**：不是 P/E 低就是便宜，理解价格与价值的真正关系
- **识别护城河**：看到一家公司时，能判断它的竞争优势是否可持续
- **建立投资纪律**：理解为什么不借债、为什么长期持有、什么时候该卖出
- **用文明视角理解财富**：跳出短期市场波动，从人类经济发展的大趋势中定位自己的投资

## 课程主线

这门课的逻辑是一条清晰的递进线：

1. **Session 1-2** 回到起点：价值投资诞生于 1930 年代大萧条，Graham 在废墟中总结出三大基石——所有权思维、市场先生、安全边际。这是整座大厦的地基。
2. **Session 3-5** 搭建估值工具箱：Greenwald 的四层估值法是把 Graham 的直觉系统化的产物。从资产价值到盈利能力价值，再到竞争优势和增长，一层层拆解"这家公司值多少钱"。
3. **Session 6** 是一次质的飞跃：巴菲特和芒格把价值投资从"捡烟蒂"升级为"以合理价格买伟大公司"。能力圈、护城河、长期持有——这些概念让价值投资从一种交易策略变成了一种商业伙伴关系。
4. **Session 7** 面对最难的敌人——自己。行为金融解释了为什么价值投资的逻辑不难但大多数人做不到。
5. **Session 8-9** 把视野拉到最大：李录从文明范式变迁的角度重新定义了财富，并把价值投资的实践延伸到中国和全球舞台。
6. **Session 10** 是最难的一讲：用最批判的眼光审视整个框架——有效市场假说、索罗斯的反身性理论、价值溢价的消失、量化革命——逼迫你真正理解价值投资的边界，而不是盲目信奉。

贯穿全课的是一个**实践者视角**：不只是讲理论，而是通过真实的投资案例——从 Graham 买 Net-Net 股票，到巴菲特持有可口可乐，到李录 22 年持有比亚迪——让你体会每一个投资决策背后的思考过程。

## Session 速览

| # | 标题 | 核心问题 |
|---|---|---|
| 1 | 价值投资的诞生 | 价值投资是什么？它为什么诞生在人类历史上最黑暗的经济时期？ |
| 2 | 三大基石：所有权、市场先生与安全边际 | Graham 留下的三个核心理念到底在说什么？ |
| 3 | 估值框架（上）：资产价值与盈利能力价值 | 一家公司到底值多少钱？ |
| 4 | 估值框架（下）：竞争优势与护城河 | 什么样的公司能长期保持超额回报？ |
| 5 | 增长与价值 | 增长什么时候创造价值，什么时候毁灭价值？ |
| 6 | 能力圈与优质公司 | 从"捡烟蒂"到"买伟大公司"的跃迁是怎么发生的？ |
| 7 | 市场心理与投资纪律 | 价值投资的逻辑不难，为什么大多数人做不到？ |
| 8 | 文明、财富与全球价值投资 | 什么是真正的财富？价值投资人应该如何定位自己？ |
| 9 | 价值投资的现代化与中国机遇 | 价值投资在中国的实践、挑战与未来 |
| 10 | 批判与辩证：反对者的声音 | 为什么有聪明人不相信价值投资？它的边界在哪里？ |

## 关于这份笔记

这份笔记整合了多个来源的公开知识：哥伦比亚商学院 Heilbrunn Center for Graham & Dodd Investing 的价值投资课程体系（Bruce Greenwald 教授创立）、Benjamin Graham 和 Warren Buffett 的经典著作与公开演讲、以及李录（喜马拉雅资本创始人）2024 年在北京大学的公开演讲。

- **语言**：中文为主，关键术语首次出现时用"中文（English）"格式标注
- **案例**：每讲都包含真实投资案例，尽量还原决策思路而非只讲结果
- **定位**：帮助快速建立价值投资的完整框架，不是教材替代品

> **一句话总结这门课**：价值投资不是一种交易技巧，而是一种理解世界的方式——理解商业如何创造价值，理解市场如何定价，理解人性如何扭曲判断，理解文明如何演进。掌握了这些，投资只是自然而然的结果。
