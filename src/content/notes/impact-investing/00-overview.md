## 这门课在学什么

"影响力投资"（Impact Investing）回答的是一个听起来很简单、但执行极其复杂的问题：**可不可以用投资来解决社会和环境问题，同时还赚钱？**

这门课从理论基础出发（什么是影响力？如何衡量？），延伸到具体的资产类别（私募股权、债券、房地产、风险投资）、机构角色（DFI、基金会、家族办公室）、以及测量框架（IRIS、IMP、GIIN），最终覆盖"如何设计和评估一个真正产生社会效益的投资组合"。

## 为什么要学

2023 年，全球影响力投资规模估计超过 1.16 万亿美元（GIIN 数据）。这不再是一个小众的"善意投资"——养老基金、主权财富基金、保险公司都在把"影响力"纳入投资框架。对于希望在金融机构从事 ESG/可持续投资、发展融资或社会创业的人，这是必要的知识基础。

## 学完之后能做什么

你能识别"影响力洗白"（Impact Washing）与真正的影响力投资的区别；能用 IMP 的五个维度框架评估一个投资项目的社会效益；理解绿色债券、社会效益债券（SIB）、开发性金融机构（DFI）的运作机制；以及在设计投资策略时，知道如何在财务回报和社会影响之间做出有依据的取舍。

## 课程主线

### Part I：基础框架（Session 1-2）

影响力投资是什么、不是什么，和 ESG 的区别在哪里。行业发展史：从微型金融到现在的万亿规模，这条路是怎么走过来的？Rockefeller 基金会 2007 年创造了"Impact Investing"这个词，之后行业经历了哪些演变？

- Session 1：影响力投资是什么——定义、历史与行业规模
- Session 2：影响力光谱——从慈善到传统投资，中间有多少种可能？

### Part II：衡量与认证（Session 3-4）

影响力最大的挑战之一是：怎么证明你真的产生了影响？IRIS 指标体系、IMP 五维框架、SDG 对齐、B 认证——每种方法解决什么问题、又有什么局限？

- Session 3：衡量影响力——IRIS、IMP 与"加法性"问题
- Session 4：B Corp 与影响力认证——商业目的与社会目标可以兼容吗？

### Part III：资本结构与工具（Session 5-7）

不同的投资工具（股权、债券、担保、混合金融）如何服务于不同的影响力目标？绿色债券、社会效益债券（SIB/DIB）、混合融资（Blended Finance）的机制和实际效果。

- Session 5：影响力股权投资——VC/PE 如何做影响力？
- Session 6：固定收益工具——绿色债券、蓝色债券与社会效益债券
- Session 7：混合融资——用公共资金撬动私人资本

### Part IV：机构角色（Session 8-10）

谁在参与影响力投资？开发性金融机构（DFI，如 IFC、DEG）、基金会（Ford、Gates）、家族办公室（单家族 vs 多家族）、主权财富基金——各自的激励结构和约束条件是什么？

- Session 8：开发性金融机构——政策性资本如何"催化"市场？
- Session 9：基金会与慈善资本——PRI 与 MRI 的策略
- Session 10：家族办公室与超高净值——影响力投资的私人资本维度

### Part V：前沿议题（Session 11-12）

气候投融资、新兴市场影响力投资的特殊挑战（数据稀缺、汇率风险、政治不确定性），以及影响力投资的未来——规模化是否必然稀释影响力？

- Session 11：气候投融资——从绿色溢价到转型风险
- Session 12：新兴市场与影响力的边界——在信息最少的地方，如何做最重要的投资？

## 关于这份笔记
