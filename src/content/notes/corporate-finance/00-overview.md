## 这门课在讲什么

想象你是一家投资银行的分析师，老板扔给你一份案例——某个公司想融资，你需要告诉他：这家公司值多少钱？它应该借多少债？用什么折现率？

这门课（INAF U6301 Corporate Finance）给你的，就是回答上述问题的**完整工具箱**。

从左到右，分析一家公司需要走五步：

| 步骤 | 内容 | 核心产出 |
|---|---|---|
| 第一步 | 历史绩效分析 | 理解公司过去怎么赚钱 |
| 第二步 | 未来现金流预测 | 预测公司未来能产生多少钱 |
| 第三步 | 最优资本结构 | 确定目标负债率（D/C） |
| 第四步 | 资本成本（WACC） | 计算折现率 |
| 第五步 | DCF 估值与可比倍数 | 得出公司股权价值 |

这五步是**高度相互依赖**的——你在第三步得出的目标负债率，会直接影响第四步的 WACC；第四步的折现率，又决定了第五步算出的价值。所以必须按顺序来。

## 全局流程图（大脑地图）

```
第一步：历史绩效分析
  └─ DuPont 公式 → ROE 拆解
       ├─ 盈利能力（NI/Sales）：毛利率、净利率、EBIT
       ├─ 效率（Sales/Assets）：资产周转率、存货天数
       └─ 杠杆（Assets/Equity）：D/C、利息保障倍数

第二步：未来现金流预测
  ├─ 预测利润表 + 资产负债表（按比例/假设）
  ├─ 计算 NWC（营运资本）和 NFA（净固定资产）
  └─ FCFF = EBIAT − ΔNFA − ΔNWC

第三步：最优资本结构
  ├─ 分析变动性（OPM 波动）→ RWC EBIT
  ├─ 分析脆弱性 → 确定 h（最低利息保障倍数）
  └─ RWC EBIT / h → Max Interest → Max Debt → D/C 目标

第四步：WACC
  ├─ 债务成本：r_D × (1 − 税率)
  └─ 股权成本：CAPM = r_f + β_E × (r_m − r_f)
       └─ β_E 通过「剥离→平均→重新加杠杆」三步法得出

第五步：DCF 估值
  ├─ Stage 1：详细预测期 FCFF 的 NPV
  ├─ Stage 2：残值（永续增长 or 竞争市场）的 PV
  ├─ + 冗余资产（超额现金等）
  ├─ − 现有债务市值
  └─ = 股权价值（MVE）
```

## 投资含义：为什么这五步很重要

这套框架不只是考试工具，在实际工作中它是**每一笔并购交易、融资决策、项目投资**的分析基础：

- **并购方**：用 DCF 判断目标公司值多少钱，用 WACC 折现
- **被并购方**：优化资本结构，让 WACC 尽可能低，以提升自身估值
- **债权人**：关注利息保障倍数，判断能不能还钱
- **股东**：关注 ROE，判断钱有没有被高效使用
