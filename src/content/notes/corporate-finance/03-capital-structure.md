## 引入：债务是一把双刃剑

借钱的好处人人知道：利息可以抵税（Tax Shield）。假设你借了 100 块，利率 10%，税率 30%，实际利息成本只有 10% × (1−30%) = 7%。国家帮你补贴了利息的 30%。

但债务也有坏处：一旦公司陷入困境，债主要你还钱，公司就面临**财务困境成本（Financial Distress Costs）**——客户跑了、员工离职、好项目做不成。

**最优资本结构**，就是在这两者之间找到平衡点——在最坏的情形下，公司依然能够偿付利息。

---

## 核心框架：三步走找到目标负债率

### 总览

```
Step 1：计算当前资本结构（理解现状）
Step 2：确定最优负债水平（核心分析）
  ├─ 确定 RWC OPM（现实最坏情景营业利润率）← 看变动性
  ├─ 确定 h（最低利息保障倍数）← 看脆弱性
  ├─ 推算 RWC EBIT → Max Interest → Max Debt
  └─ 在不同评级假设下做 Rating 迭代，找到自洽解
Step 3：再资本化（Recapitalization）
  └─ 调整到目标负债，注意总资本不变
```

---

## 第一步：理解当前资本结构

**BV vs MV 的 D/C 计算**

评级机构（Moody's、S&P）用**账面价值（Book Value）**评级，因为市场价格波动太大、容易被操控。

```
D/C @BV = Debt / (Debt + BV Equity)
D/C @MV = Debt / (Debt + MV Equity)
```

**以课堂练习题为例（Sales=100，Debt=50，BV Equity=50，MV Equity=150）**：

|  | Capital | D/C | D/E |
|---|---|---|---|
| @BV | 50+50=100 | 50/100=50% | 50/50=100% |
| @MV | 50+150=200 | 50/200=25% | 50/150=33% |

## 第二步：找最优负债量——七步法

### 步骤一：计算当前 OPM

```
OPM = EBIT / Sales = 18 / 100 = 18%
```

### 步骤二：评估变动性，确定 RWC OPM

**变动性（Variability）** 是指利润率在最坏情景下会下降多少。

分析思路：
- 历史波动有多大？（看过去最差年份的 OPM）
- 行业竞争格局是否稳定？（越稳定，越难有大波动）
- 产品/服务的刚需程度？（刚需越高，OPM 越稳定）

如果案例说「RWC OPM = 当前水平的 1/3」：

```
RWC OPM = 18% × (1/3) = 6%
RWC EBIT = RWC OPM × Sales = 6% × 100 = 6
```

### 步骤三：评估脆弱性，确定 h

**脆弱性（Vulnerability）** 决定公司需要多高的「利息保险」。脆弱性高的公司，一旦遇到财务困境，损失更大，所以需要更高的保障倍数（h）。

| 降低脆弱性（可以借更多） | 提高脆弱性（要少借） |
|---|---|
| 资产以有形资产为主 | 依赖关键人才/定制化供应商 |
| 未来无重大资本支出计划 | 计划大规模扩张/收购 |
| 产品无需售后长期支持 | 销售耐用品，客户依赖售后 |
| 非竞争性行业 | 激烈竞争，竞争对手会趁乱抢份额 |

典型参考区间（仅供参考，实际需结合案例判断）：

```
h = 2.0x → 不太脆弱（如快消品公司 Wrigley）
h = 4.0x → 非常脆弱（如重型制造 Massey Ferguson）
```

### 步骤四：计算最大利息 & 目标债务

```
Max Interest = RWC EBIT / h = 6 / 2 = 3
Max Debt = Max Interest / Interest Rate = 3 / 8% = 37.5
```

**注意**：这里的 Interest Rate 用的是公司当前承担的债务利率（由信用评级决定）。

### 步骤五：计算目标 D/C 比率

```
D/C @BV = Max Debt / Capital@BV = 37.5 / 100 = 37.5%
D/C @MV = Max Debt / Capital@MV = 37.5 / 200 = 18.75%
```

> **关键提示**：在再资本化过程中，总资本（Capital）保持不变！增加债务 = 同等减少股权（回购），资本总额不动。

### 步骤六：评级迭代——找到自洽解

这是最容易犯错的地方。

你在步骤四假设利率是 8%（BBB 评级），但算出来的 D/C 37.5% 对应的实际评级是 AA（利率 6%），二者不一致——这叫 **Mismatch（不匹配）**。

解决方法：对每个评级逐一试算，找到「假设评级 = 实际对应评级」的那个点。

**完整迭代表（Max Interest = 3 不变）**：

| 假设评级 | 利率 | D* = 3/利率 | D/C @BV | 对应实际评级 | 是否匹配？ |
|---|---|---|---|---|---|
| AAA | 5% | 60 | 60% | BB | ✗ |
| AA | 6% | 50 | 50% | BBB | ✗ |
| **A** | **7%** | **42.86** | **42.86%** | **A** | **✓ 匹配！** |
| BBB | 8% | 37.5 | 37.5% | AA | ✗ |
| BB | 10% | 30 | 30% | AA | ✗ |

**结论**：目标评级是 A，利率 7%，目标债务约 43。

### 步骤七：计算实际利息保障倍数（Sanity Check）

```
Actual Coverage = Actual EBIT / Max Interest = 18 / 3 = 6.0x
```

注意：这是用真实 EBIT 算的（不是 RWC EBIT），用来向评级机构展示正常情况下的保障能力。

## 第三步：再资本化

确定了目标债务后，公司需要调整资本结构：

```
@MV（市场价值口径）：

         Before   Change   After
Debt     31,376   -2,260   29,116  ← Optimal Debt
Equity   29,301   +2,260   31,561
Capital  60,677    0       60,677  ← 保持不变！
```

减少负债 = 回购股票（Less Debt, More Equity）  
增加负债 = 发行新债回购股票（More Debt, Less Equity）

## 案例二：更复杂的实战练习

**给定数据**：Sales = 5,000，EBIT = 650，OPM RWC = 7%，h = 3

**Step 1**：RWC EBIT = 5,000 × 7% = 350；Max Interest = 350 / 3 = 117

**Step 2**：对每个评级试算 D*：

| 评级 | 利率（30年期） | D* | D/C @BV（Capital=5,000） |
|---|---|---|---|
| AA | 4.31% | 2,707 | 54.1% |
| A | 4.45% | 2,622 | 52.4% |
| **BBB** | **4.99%** | **2,338** | **46.8%** |
| BB | 6.90% | 1,691 | 33.8% |
| B | 7.96% | 1,466 | 29.3% |

对照评级标准表（BBB 对应 D/C @BV ≈ 46.4%），BBB 行的 D/C = 46.8% ≈ 46.4%，基本匹配 → **最优评级 BBB**。

---

## 核心结论

> 最优资本结构分析的本质是：**在最坏的现实情景下，公司能用多少 EBIT 支付多少利息，再反推出能承受多少债务**。这个框架把「能借多少」变成了一道有答案的计算题，而不是拍脑袋的判断。
