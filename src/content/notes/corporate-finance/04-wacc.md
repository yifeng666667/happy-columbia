## 引入：折现率从哪里来？

假设一家公司明年的现金流是 100 块，你应该为它付多少钱？如果你要求 10% 的回报，你会付 100/1.1 = 90.9 块。

这个「10%」就是折现率——它代表了投资者对这类风险资产所要求的**最低回报**。

公司的资金来自两个来源：债权人和股东。他们的要求回报率不同，我们需要按比例加权，得出**加权平均资本成本（WACC）**：

```
WACC = (1 − t) × (D / Capital) × r_D  +  (E / Capital) × r_E
       └─ 税后债务成本 × 债务权重 ─┘    └─ 股权成本 × 股权权重 ─┘
```

其中：
- **t** = 企业所得税率
- **D / Capital，E / Capital** = 来自第三步的**目标**资本结构（用市值计算）
- **r_D** = 债务利率（对应目标评级的利率）
- **r_E** = 股权资本成本（用 CAPM 计算）

---

## 债务成本（r_D）

债务成本很直接：**用公司目标评级对应的利率**（来自第三步的结论）。

记得乘以 (1 − 税率)，因为利息支出可以抵税：

```
税后债务成本 = r_D × (1 − 税率)
```

例：BBB 评级，利率 4.99%，税率 30%  
→ 税后债务成本 = 4.99% × (1 − 30%) = **3.5%**

---

## 股权成本（r_E）：CAPM 模型

股权成本更复杂，因为股票没有约定利率。我们用 **CAPM（资本资产定价模型）**：

```
r_E = r_f  +  β_E × (r_m − r_f)
      无风险  └─ 股权 Beta ─┘ └─ 市场风险溢价 ─┘
      利率
```

三个输入变量：

| 变量 | 含义 | 如何取值 |
|---|---|---|
| r_f（无风险利率） | 国债利率，代表「没有风险」的回报 | 用**最长期**可用国债（30年），因为要与投资期限匹配 |
| r_m − r_f（市场风险溢价，MRP） | 股票市场历史平均超额回报 | 通常由题目给出，历史数据约 5% |
| β_E（股权 Beta） | 公司股票相对市场的系统性风险 | **需要通过可比公司推算**，见下文 |

---

## 核心难点：如何得到 β_E（股权 Beta）

公司自身的 β_E 不能直接用历史数据，因为历史数据反映的是**当前**资本结构下的风险。当资本结构改变（再资本化）后，β_E 也会变。

正确做法：**三步走「剥离-平均-重新加杠杆」**

### Step 1：找可比公司（Comparables）

选取与目标公司业务相似的上市公司，获取它们的历史股权 Beta（β_E_comp）。

### Step 2：剥离杠杆，得到资产 Beta（β_A）

股权 Beta 包含了公司借债的风险，不同公司资本结构不同，不能直接比较。需要「剥离」掉财务杠杆：

```
β_A = β_E × E / (E + D_net)

其中：
  E = 股权市值（MV Equity）
  D_net = 净债务（可以用总债务，或「债务 − 超额现金」）
```

对所有可比公司做这个计算，然后**取平均**，得到行业平均资产 Beta（β_A_avg）。

### Step 3：按目标资本结构重新加杠杆（Re-lever）

用目标资本结构（来自第三步）重新给 β_A 加上杠杆：

```
β_E_target = β_A × (1 + D_target / E_target)
```

这样得到的 β_E，才是公司在**目标资本结构**下应有的股权风险系数。

## 完整 WACC 计算案例

**给定条件**（来自课堂 Exercise 2）：
- 历史观测 β_E = 0.9，当前 Debt = 2,000，MV Equity = 6,000
- 目标资本结构：BBB 评级，D* = 2,338，E* = 5,662（= 8,000 − 2,338）
- 无风险利率 r_f = 3.35%（30年期国债），MRP = 5%
- 税率 = 30%，债务成本 r_D = 4.99%

**Step 5：计算目标资本结构下的 β_E**

```
剥离（Unlever）：
  β_A = 0.9 / (1 + 2,000/6,000) = 0.9 / 1.333 = 0.675

重新加杠杆（Relever）：
  β_E = 0.675 × (1 + 2,338/5,662) = 0.675 × 1.413 = 0.954 ≈ 0.95
```

**Step 6：计算 WACC**

```
r_E = 3.35% + 0.95 × 5% = 3.35% + 4.75% = 8.1%

税后 r_D = 4.99% × (1 − 30%) = 3.49% ≈ 3.5%

权重：
  D/Capital @MV = 2,338 / (2,338 + 5,662) = 29.2%
  E/Capital @MV = 5,662 / 8,000 = 70.8%

WACC = 29.2% × 3.5% + 70.8% × 8.1%
     = 1.02% + 5.73%
     = 6.75% ≈ 6.77%
```

## 常见错误提醒

| 错误做法 | 正确做法 |
|---|---|
| 用账面价值权重计算 WACC | 用**市场价值**权重 |
| 用历史 β_E 直接代入 CAPM | 先剥离，取均值，再用目标结构重新加杠杆 |
| 用短期国债利率作为 r_f | 用**最长期**可用国债（匹配投资期限） |
| 忘记税盾，不乘 (1−t) | 债务成本必须是税后成本 |
| 用「当前」资本结构权重 | 用「**目标**」资本结构权重 |

---

## 核心结论

> WACC 是整个估值框架的定价之锚——它太高，公司估值就低；它太低，估值就高。因此，合理选择可比公司、谨慎处理 Beta 的剥离与再加杠杆，是这一步最重要的技术判断。WACC 不是一个「查查就有」的数字，而是需要你对公司业务风险和资本结构有深刻理解后才能算出来的结果。
